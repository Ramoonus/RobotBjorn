# Robot Bjorn Compatibility #

## Arduino ##
### Confirmed working ###

* Arduino Uno R3 (primairy goal)
* Arduino Leonardo R3
* Arduino Ethernet
* [Sparkfun RedBoard](https://www.sparkfun.com/products/11575)

### Untested ###
Untested but possibly working:

* Arduino Due
* Arduino Mega 2560
* Arduino Leonardo
* Arduino ADK
* Arduino Micro
* Arduino Nano
* Arduino Pro Mini
* Arduino Pro
* Arduino Fio

### Incompatible ###

* Arduino Esplora
* Arduino LilyPad
